# 🌳 Conversation Tree English

Learn everyday English by *reading and practicing real conversations* — like browsing
a library of chats instead of memorizing sentences. The goal is simple:
**become able to talk about yourself in English.**

This is **not** an AI chatbot. It's a **Conversation Story** platform. Each topic holds
many complete conversations (stories). You read one like a LINE chat, then practice
speaking your side of it.

## How it works

```
Topic  →  Story (a complete conversation)  →  Read  →  Practice  →  (Continue, coming soon)
```

- **📖 Read mode** — the conversation already exists. Read it through, tap 🔊 to hear it,
  toggle the Japanese translation. No typing required.
- **🎤 Practice mode** — the same conversation replays with *your* lines hidden. The AI
  asks, and you answer by tapping a graded suggestion (Beginner / Intermediate / Advanced)
  or typing your own. Finish the story to complete it.
- **Continue mode** *(coming soon)* — keep chatting freely from where the story ended.
  This needs an LLM, so it's stubbed for a later AI pass.

## Features

- **Story Library** — browse conversations Netflix-style, grouped by topic.
- **Conversation tree** (left) — the current story rendered as a path; the active node is
  highlighted and unreached nodes are dimmed.
- **Progress** (right) — per-topic and overall *stories completed*, persisted in `localStorage`.
- **Japanese translation** — one-tap per line, plus a global "Show JP".
- **Voice** 🔊 — Web Speech API reads any line aloud.
- **Dark mode** — manual toggle, remembered.

All content is **static JSON, personalized to the user's profile** (Nissan engineer, into
AI / autonomous driving, wants a PhD abroad, likes KFC, YouTube, Steam, Tesla). No backend
or API keys are needed for this build.

## Data model (UI-independent)

Conversation data lives entirely in `src/data/` and is decoupled from the UI, so it can
scale to thousands of stories or be generated by an LLM later.

```ts
Story = {
  id, topicId, title, title_ja, emoji, difficulty, summary,
  messages: StoryMessage[]            // the full conversation
}
StoryMessage = {
  id, speaker: "ai" | "user", en, ja, voice?,
  suggested?: { beginner, intermediate, advanced }   // used in Practice (user lines)
}
Topic = { id, emoji, title_en, title_ja, description, color }
```

  context/AppContext.tsx   state, Read/Practice engine, progress
  data/topics.ts        topic categories
  data/stories.ts       the conversation stories (static, personalized)
  lib/types.ts          Story / StoryMessage / SuggestedAnswers / Topic
  lib/speech.ts         speech-synthesis helper
```

## Adding a story

Append a `Story` object to `src/data/stories.ts` with `topicId` pointing at a topic in
`src/data/topics.ts`. Alternate `ai` / `user` messages; give each `user` line a `suggested`
trio. The library, tree, chat, and progress all derive from this automatically.
